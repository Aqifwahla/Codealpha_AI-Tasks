## FAQ Chatbot using NLP

### Description
This project is a simple FAQ chatbot developed using Python and Natural Language Processing techniques.

### Features
- User-friendly GUI
- NLP-based question matching
- TF-IDF Vectorizer
- Cosine Similarity
- JSON knowledge base

### Technologies Used
- Python
- Scikit-learn
- Tkinter
- JSON

### Installation

```bash
pip install -r requirements.txt
python chatbot.py
```
